# Frequency Range Reference Guide

Complete guide to interpreting spectral analysis results by mapping frequency ranges to instruments, vocals, and tonal characteristics.

## Vocal Frequency Ranges

### Male Vocals
- **Fundamental Range:** 85-180 Hz
- **Harmonic Extension:** 100 Hz - 8 kHz
- **Presence/Clarity:** 2-5 kHz

### Female Vocals  
- **Fundamental Range:** 165-255 Hz
- **Harmonic Extension:** 200 Hz - 12 kHz
- **Presence/Clarity:** 2-5 kHz

### Interpretation for Analysis
- Mean pitch 100-180 Hz + harmonics to 8 kHz → Likely male vocal
- Mean pitch 180-255 Hz + harmonics to 12 kHz → Likely female vocal
- Mean pitch 400-600 Hz with wide bandwidth → Possibly dual male/female vocals
- Multiple peaks in same vocal range → Choir/backing vocals

## Frequency Band Descriptors (Tonal Quality)

### Sub Bass (20-60 Hz)
- **Rumble:** 20-31.5 Hz
- **Chest/Thump:** 40-63 Hz
- **Character:** Felt more than heard, physical impact

### Bass (63-250 Hz)
- **Bottom:** 63-80 Hz
- **AC/Ground:** 80-100 Hz (electrical hum range)
- **Warm:** 100-125 Hz
- **Boom/Punch:** 125-200 Hz
- **Character:** Fundamental weight, power

### Midrange (250-2000 Hz)
- **Fullness/Mud:** 250-500 Hz
- **Middle C (A440):** 315-630 Hz
- **Honk:** 500-800 Hz
- **Whack:** 630-1250 Hz
- **Character:** Core tonal body, presence

### High Mids (2-6 kHz)
- **Tinny:** 1-2 kHz
- **Crunch:** 2-3.15 kHz
- **Character:** Clarity, definition, articulation

### High Frequencies (6-20 kHz)
- **Sibilance:** 5-8 kHz (s, sh, z sounds)
- **Air:** 8-10 kHz
- **Definition:** 10-12.5 kHz
- **Edge:** 12.5-16 kHz
- **Treble:** 16-20 kHz
- **Pierce:** 10-20 kHz (can cause listener fatigue)
- **Character:** Sparkle, brilliance, air

## Practical Analysis Guidelines

### Identifying Instruments in Spectral Analysis

**Bass-Heavy Mix (40-200 Hz dominant):**
- Check for sustained energy → Bass guitar/synthesizer
- Check for rhythmic pulses → Kick drum
- Check for melodic movement → Bass instrument

**Mid-Range Activity (200-2 kHz):**
- Harmonic complexity + vocal formants → Vocals
- Rich harmonics + warmth → Strings/acoustic instruments
- Clean sine-like waves → Synthesizers
- Sustained with vibrato → Brass/woodwinds

**High-Frequency Content (8-20 kHz):**
- Sustained shimmer → Cymbals/hi-hats
- Burst patterns → Percussion attacks
- Continuous with sibilance → Vocals (s, sh, z sounds)
- Airy quality → Flutes, breath sounds

### Gender Identification in Vocals

**Male Vocalist Indicators:**
- Fundamental frequency: 85-180 Hz
- Energy concentrated in 100-500 Hz range
- Harmonics extending to ~8 kHz
- Formants in lower mid-range

**Female Vocalist Indicators:**
- Fundamental frequency: 165-255 Hz  
- Energy concentrated in 200-800 Hz range
- Harmonics extending to ~12 kHz
- Formants in upper mid-range

**Dual Vocalist Detection:**
- Wide spectral bandwidth (3000+ Hz)
- Two distinct energy centers in vocal range
- Mean pitch in overlap region (400-600 Hz)
- Harmonic complexity from multiple sources

**Choir Detection:**
- Multiple simultaneous peaks in same frequency band
- Rich harmonic density
- Reduced dynamic range (voices average together)
- Smooth spectral energy distribution

### Intensity/Aggression Indicators

**Aggressive Mix Characteristics:**
- High sibilance (>0.3) → Cutting, intense vocal delivery
- Strong 2-4 kHz presence → Crunch, attack
- Sharp transients → Percussive emphasis
- Compressed dynamics → Sustained intensity

**Smooth/Mellow Characteristics:**
- Low sibilance (<0.2) → Soft vocal delivery
- Reduced 2-4 kHz → Less aggressive
- Gentle transients → Flowing feel
- Wide dynamics → Expressive variation

## Using This Reference

When analyzing spectral data:

1. **Identify Peak Frequencies:** Where is energy concentrated?
2. **Map to Instrument Ranges:** Use the tables above
3. **Check Bandwidth:** Wide = multiple sources, narrow = focused
4. **Examine Harmonics:** Extension indicates instrument type
5. **Note Temporal Patterns:** Sustained vs percussive
6. **Cross-reference Metrics:** Combine frequency + intensity + temporal data

This allows translation of mathematical spectral analysis into musical interpretation.
